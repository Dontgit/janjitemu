# UI / UX Direction — Temujanji

## Prinsip UI
UI Temujanji harus:
- sederhana
- cepat dipahami UMKM
- tidak terlalu teknis
- mobile-friendly
- fokus pada aksi utama

## Karakter UI
- clean
- modern
- hangat
- profesional tapi tidak kaku

## Gaya Visual
- banyak whitespace
- card-based layout
- warna utama yang menenangkan
- typography sederhana dan jelas
- ikon seperlunya

## Rekomendasi Warna
### Opsi 1 — Profesional & lembut
- Primary: biru lembut
- Secondary: slate / abu modern
- Accent: hijau success

### Opsi 2 — Hangat & lokal
- Primary: teal atau hijau kebiruan
- Secondary: krem terang / abu muda
- Accent: oranye lembut

## Komponen UI Penting
- top navigation sederhana
- sidebar dashboard ringan
- card booking summary
- calendar / schedule panel
- modal reschedule
- form booking step-by-step
- badge status booking

## UX Booking Page
Booking page publik harus bisa dipahami dalam hitungan detik.
Urutan ideal:
- pilih layanan
- pilih tanggal
- pilih jam
- isi data diri
- konfirmasi booking

## UX Dashboard Owner
Dashboard owner harus fokus ke:
- apa booking hari ini
- siapa yang belum confirm
- slot mana yang padat
- apa yang perlu ditindak

## Mobile Experience
Karena banyak owner UMKM buka dashboard dari HP:
- dashboard harus usable di mobile
- table jangan terlalu kompleks
- gunakan card/list untuk versi mobile
- booking page harus mobile-first

## Tone Visual Brand
Temujanji sebaiknya terasa:
- ramah
- praktis
- terpercaya
- membantu usaha jadi lebih rapi

## Hal yang Harus Dihindari
- UI terlalu enterprise
- terlalu banyak menu di awal
- terlalu banyak warna mencolok
- flow booking terlalu panjang
- istilah teknis yang membingungkan user
