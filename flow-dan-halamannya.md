# Flow dan Halaman — Temujanji

## Flow Utama Customer

### Flow Booking
1. Customer membuka link booking bisnis
2. Customer memilih layanan
3. Customer memilih tanggal
4. Customer memilih jam yang tersedia
5. Customer mengisi data diri
6. Customer submit booking
7. Sistem menampilkan status booking
8. Customer menerima konfirmasi / reminder

## Flow Utama Owner

### Flow Setup Awal
1. Owner daftar akun
2. Owner membuat profil bisnis
3. Owner menambahkan layanan
4. Owner mengatur jam operasional
5. Owner membagikan link booking

### Flow Operasional Harian
1. Owner login
2. Owner melihat dashboard booking hari ini
3. Owner mengonfirmasi booking pending
4. Owner melakukan reschedule jika diperlukan
5. Owner menandai booking selesai / no-show / batal
