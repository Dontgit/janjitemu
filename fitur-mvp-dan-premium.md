# Fitur MVP dan Premium — Temujanji

## Fitur MVP

### 1. Landing Page Sederhana
- penjelasan produk
- CTA daftar
- CTA coba demo booking

### 2. Registrasi Bisnis
- daftar akun owner
- buat nama bisnis
- pilih kategori bisnis

### 3. Setup Layanan
- tambah layanan
- durasi layanan
- harga opsional
- deskripsi singkat

### 4. Jam Operasional
- set hari kerja
- set jam buka/tutup
- set slot per hari
- buffer antar booking

### 5. Booking Page Publik
- halaman publik per bisnis
- pilih layanan
- pilih tanggal
- pilih jam tersedia
- isi data customer
- kirim booking

### 6. Dashboard Booking
- daftar booking hari ini
- booking mendatang
- booking berdasarkan status
- filter tanggal

### 7. Status Booking
- pending
- confirmed
- completed
- cancelled
- no-show
- rescheduled

### 8. Reschedule oleh Admin
- ubah tanggal/jam booking
- update status booking

### 9. Customer List Sederhana
- nama
- nomor kontak
- riwayat booking singkat

### 10. Reminder Basic
- pengingat internal di dashboard
- email reminder sederhana

## Fitur Premium

### 1. Multi Staff
- pilih staff saat booking
- jadwal per staff
- blok jadwal per staff

### 2. Custom Branding
- logo bisnis
- warna brand
- custom booking page appearance

### 3. Custom Domain
- contoh: booking.namabisnis.com

### 4. Reminder Otomatis Lanjutan
- WhatsApp reminder
- email reminder multi tahap
- reminder H-1 dan H-2 jam

### 5. Deposit / DP Booking
- booking dengan pembayaran awal
- status paid / unpaid

### 6. Laporan dan Analytics
- total booking per periode
- layanan paling populer
- no-show rate
- jam ramai
- performa staff

### 7. Form Booking Kustom
- field tambahan sesuai bisnis
- catatan khusus customer

### 8. Kupon / Promo
- kode promo
- diskon layanan tertentu
- promo slot sepi

### 9. Booking Ulang Cepat
- repeat booking untuk customer lama

### 10. Multi Cabang
- beberapa lokasi bisnis
- jadwal per cabang
- staff per cabang

## Rekomendasi Paket

### Basic
- 1 bisnis
- booking page
- layanan
- jam operasional
- dashboard booking

### Pro
- semua fitur Basic
- multi staff
- custom branding
- reminder lanjutan
- laporan dasar

### Business
- semua fitur Pro
- multi cabang
- analytics lanjutan
- custom domain
- fitur operasional lebih lengkap
